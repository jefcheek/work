1. Download and install python 3.11.5 from https://www.python.org/downloads/

2. Using command prompt, check Python installed successfully
python --version

3. Using command prompt, install
pip install robotframework
pip install --upgrade robotframework-seleniumlibrary
pip install  pyYAML

4. Make sure robotframework, robotframe-seleniumlibrary, pyYAML are installed
pip list

5. Add the following extensions in Visual Studio Code. Alternatively, extensions can be added downloaded from https://open-vsx.org/
RobotCode
RobotFramework Language Server
ms-python.python



Get Chrome driver (depending on the Chrome version used) from
https://chromedriver.chromium.org/downloads/version-selection
Unzip the file and place chromedriver.exe in the same folder as the robot script
